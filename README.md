# 🎓 Predicting Analysis of Student Dropout in Academic Institutions Using Customer Churn Analysis

## 📌 Project Overview

This project leverages Customer Churn Analysis techniques and Machine Learning algorithms to predict student dropout risk in academic institutions. By analyzing student-related data such as academic performance, attendance, and engagement metrics, the model identifies students who may be at risk of discontinuing their studies. The insights generated help educational institutions take proactive measures to improve student retention and academic success.

## 🎯 Objectives

* Predict student dropout risk using Machine Learning.
* Identify key factors contributing to student attrition.
* Generate actionable insights for academic institutions.
* Support data-driven decision-making to enhance student retention.

## 🚀 Key Features

* 📊 Data Cleaning and Preprocessing

* 📉 Student Risk Identification
