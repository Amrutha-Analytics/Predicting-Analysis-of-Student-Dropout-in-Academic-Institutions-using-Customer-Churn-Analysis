#🎓 Predicting-Analysis-of-Student-Dropout-in-Academic-Institutions-using-Customer-Churn-Analysis
Predicted student dropout risk using customer churn analysis and visualized key insights through interactive Power BI dashboards to support student retention.


📌 Project Overview

This project applies Customer Churn Analysis techniques to predict student dropout risk in academic institutions. By analyzing student-related data, the model identifies students who are likely to discontinue their studies, helping institutions take proactive measures to improve retention and academic success.

🎯 Objectives
Predict student dropout risk using Machine Learning.
Identify key factors contributing to student attrition.
Generate actionable insights for academic institutions.
Support data-driven decision-making for student retention.
